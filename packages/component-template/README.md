# 说明

基于 dumi2 的组件文档站点

## 下载

```bash
npx @enode/create-project download @enode/component-template
```

## 环境准备

确保正确安装 [Node.js](https://nodejs.org/en) 且版本为 `18+`。

## 操作

```bash
# 安装依赖
yarn

# 启动本地调试
yarn start
```
